Git - отдельная программа выполняющая функции системы контроля версий
взаимодействие с которой производиться через командную строку (терминал). Git
используется для хранения и формирования последовательной истории изменений в
которой фиксируется точки сохранений (commit), каждая такая точка хранит
информацию об авторе изменений, дата/времени и непосредственно сами изменения.
Разработчик самостоятельно решает в какой может времени и какие изменения он
хочет сохранить в истории. Каталог относительно которого отслеживаются
изменения и храниться история называется репозиторием, весь репозиторий
хранится в папке с названием `.git`, при её удалении вся история будет утеряна.

`git init` - инициализирует новый пустой репозиторий в той папке относительно
которой выполняется команда.

`git status` - выводит информации об актуальном состоянии репозитория на момент
выполнения команды. Выводит информацию о файлах, изменения по которым
отслеживаются и не отслеживаюьтся, о ветке на которой находится репозиторий и в
целом его состоянии.

`git add <files>` - добавляет файл для его дальнейшего включения в точку
сохранения (commit), при создании точки сохранения файл будет записан в том
состоянии в котором он был на момент выполнения команды git add.
